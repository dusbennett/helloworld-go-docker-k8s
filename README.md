# helloworld-go-docker-k8s



##Prerequisites

Homebrew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

go
```
brew install go
```

godep
```
brew install dep
```

docker
```
brew install docker
```

docker-machine
```
brew install docker-machine
```

VirtualBox
```
brew cask install virtualbox
```
